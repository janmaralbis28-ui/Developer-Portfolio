PROFILE PHOTO INSTRUCTIONS
==========================

Your profile photo (Seated_Portfolio_2.png) needs to be placed here:
  assets/images/profile.png

Steps:
1. Copy your photo "Seated_Portfolio_2.png" to this folder
2. Rename it to: profile.png
3. Open index.html — the portrait section already references this file

The hero portrait in index.html uses:
  src="assets/images/profile.png"

The about page portrait uses:
  src="../assets/images/profile.png"

Once you place the image here, both pages will display your real photo automatically.
